# assets.zip 同梱アセット

このフォルダには、体操選手HPモックアップから実装用に整理した画像素材を格納しています。

## 方針
- ヒーロー、プロフィール、ギャラリー主要写真、問い合わせ用リング背景、スポンサー背景は高解像度で再生成しています。
- サムネイル系の小さな画像（ニュースや一部ブログ）は、モックアップから切り出した上でアップスケール・シャープ補正しています。
- 動画サムネイルは、再生成した高解像度画像から派生させています。

## ディレクトリ
- `ui/` ロゴ画像
- `hero/` ヒーロー画像
- `profile/` プロフィール写真
- `news/` 最新情報サムネイル
- `gallery/` 写真ギャラリー
- `videos/` 動画サムネイル
- `blog/` ブログカード画像
- `support/` 応援・スポンサー背景
- `contact/` お問い合わせ用画像

## 補足
- `manifest.json` に各ファイルの生成種別（再生成 / 切り出し補完 / 派生）を記載しています。
- 実運用でさらに厳密なブランド/本人再現が必要な場合は、元写真や正式ロゴに差し替えてください。
